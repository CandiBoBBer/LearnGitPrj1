##Команды
1. git init
2. git add (. or filename.ext)
3. git commit -m "Комментарий"
4. git log
5. git remote add (name: origin) (URL: https://github.com/.../projname.git)
6. git branch -M main
7. git push -u (name: origin) (branch: main or ~~master~~)
